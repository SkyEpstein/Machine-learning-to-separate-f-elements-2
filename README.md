# README — Machine-learning-to-separate-f-elements-2

This repository predicts logD (the log of the distribution coefficient) for f-element solvent extraction and provides confidence estimates and calibrated uncertainty ranges.

Reorganized layout (branch: repo-reorg)
- data/             — data/data.zip (unzipped into data/ produces Training_Data_V27.csv and Testing_Data_V39.csv)
- src/              — lightweight wrappers to run the top-level scripts (keeps original scripts at root for now)
- figures/          — figures generated by the pipeline (kept tracked)
- results/          — generated CSVs, Excel workbook, slide decks (added to .gitignore by default)
- docs/             — long-form write-ups and methods

Quick start
1) Install requirements:
   pip install -r requirements.txt
2) Unpack data.zip into data/:
   unzip data/data.zip -d data/
3) Run a script via the src/ wrappers, e.g.:
   python3 src/confidence_tune.py
   python3 src/deploy_final.py
   python3 src/make_figures.py

To run the full pipeline:
   ./run.sh all
